Как запустить:
1) Скачать Meshroom-2021.1.0
2) установить python
3) Запустить в виртуальном окружении venv (.\Diplom\Diplom)
4) Во вкладке настройки указать путь к ДИРЕКТОИИ с meshroom.exe ("D:\Meshroom-2021.1.0")
5) переместить в главное окно папку с фотографиями одного формата (jpg,png)
6) запустить
